Documentation

Variables

The Tyfy Theme for Bootstrap introduces numerous new or updated SASS variables.

Grid

A new grid unit value governs margins, paddings, and line-heights in the theme.

$tyfy-grid-unit
This is a rem-unit value that should be set to half of the default line height. As this is a Tyfy variable, a custom value for the variable should be add at the very top of the main stylesheet.
$spacer
This Bootstrap variable has been set to 2 * $tyfy-grid-unit.

Colors

Several new colors have been added to Bootstrap's $colors SASS map. These colors can be accessed in SASS using the color functions provided in Bootstrap. The additional colors come from the Columbia College web style guide.

$blue-cc
#66aee6; // columbia college blue, p292
$blue-ca
#418fde; // college alumni blue, p279
$blue-cu
#b9d9eb; // columbia university blue, p290
$blue-cd
#002855; // college dark blue, p295
$gold-cc
#fcd450; // college gold, p122
$gray-wl
#cdc9c5; // college warm grey light, p3
$gray-wd
#5e514e; // college warm grey dark, p11
$blue-cu2
#a8cee2; // columbia university blue 2, p291
$blue-cu3
#75aadb; // columbia university blue 3, p284
$blue-cu4
#0038a8; // columbia university blue 4, p286
$blue-ad
#0060a8; // cc-seas admissions blue, p300
$cyan-ad
#0099e6; // cc-seas admissions cyan
