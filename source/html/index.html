<%include head.html%> Tyfy Theme for Bootstrap <%include header.html%>

This customization of Bootstrap v4 applies the the modular typographic scale, vertical rhythm, and intelligent margins from the Tyfy framework to Bootstrap's robust library of content and user interface patterns.

Modules from the Tyfy framework are used to amend Bootstrap in a maintainable way, consistent with Bootstrap's theming guide. Bootstrap has also been customized by editing existing variables and adding new ones to provide additional font and color options.

While some styles have been added, overrides to Bootstrap style definitions have been limited to the minimum necessary. The overrides are most often used in cases where editing Bootstrap's default variables was not enough to create the desired results.

To see the difference between Tyfy Bootstrap and regular Bootstrap, use the "Theme Toggle" option in the main menu.
