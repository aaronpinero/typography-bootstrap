Tyfy Bootstrap

Tyfy Bootstrap (TyBs) is a themed version of Bootstrap 4. The tools for modular scale, baseline grid, and intelligent margins from the Tyfy framework are combined with Bootstrap's grid system and user interface components.

TyBs has been created according to Bootstrap's theming guide. Modules from the Tyfy framework add variables and functions that are used to help customize Bootstrap's SASS variables.

TyBs overrides some Bootstrap styles to help Tyfy blend harmoniously with Bootstrap's components. The result is theme that can be applied to an existing Bootstrap 4 project with little change to the HTML structure or class attributes.

Get Started

To see the difference between Tyfy Bootstrap and regular Bootstrap, use the "Theme Toggle" option in the main menu.
