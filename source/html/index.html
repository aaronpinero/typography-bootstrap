Tyfy Bootstrap is a Bootstrap 4 theme that applies the modular scale, baseline grid, and intelligent margins from the Tyfy framework to Bootstrap's grid system and user interface components.

Tyfy Bootstrap has been created according to Bootstrap's theming guide. Modules from the Tyfy framework add values and functions that are used to help customize Bootstrap's SASS variables.

Overrides of Bootstrap styles have been limited to the minimum necessary. The overrides are most often used in cases where editing Bootstrap's default variables was not enough to create the desired results.

The result? A theme that can be applied to an existing Bootstrap 4 project with little or no change to the HTML structure or class attributes.

Get Started

To see the difference between Tyfy Bootstrap and regular Bootstrap, use the "Theme Toggle" option in the main menu.
