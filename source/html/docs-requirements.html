Requirements

Bootstrap JavaScript

Bootstrap's JavaScript-enabled components depend on the following libraries, as described in the Bootstrap startup documentation:

  • jQuery slim build
  • Popper.js

Web Font Loader

The webfontloader is a Javascript library for working with web font services. I particularly like it because it provides:

  • a way to subset fonts from Google Fonts;
  • an option to timeout a load if it takes too long; and
  • CSS state classes indicating whether a font is loaded.

Material Icons

As per the Bootstrap extensions documentation, Tyfy Bootstrap uses the Material Icons web font from Google. This font is served from Google Fonts, making it an easy addition to the theme.

  • Bootstrap icons documentation
  • Material Icons documentation
  • Material Icons on Github
