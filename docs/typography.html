Typography

Documentation and examples for typography in the Tyfy-Bootstrap theme, including global settings, headings, body text, lists, and more.

Global settings

Bootstrap sets basic global display, typography, and link styles. The Tyfy theme improves on these styles by providing variables, styles, and functions that help establish a proper typographic scale and a consistent vertical rhythm.

  • Provide an override of the Bootstrap font stack for user- or project-specific fonts.
  • For a proper typographic scale, type sizes are proportionally related.
  • Use the $font-family-base, $font-size-base, and $line-height-base attributes as our typographic base applied to the <body>.
  • Set the global link color via $link-color and apply link underlines only on :hover.
  • Use $body-bg to set a background-color on the <body> (#fff by default).
