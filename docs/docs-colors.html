Colors

Custom themes have been defined for the Tyfy theme. These additional themes can be used in the same way as Bootstraps default themes (primary, secondary, etc.). The theme colors come from various style guides used at Columbia University in the City of New York.

Custom themes
  •  
    CU, based on the web equivalent for Pantone 290C
  •  
    CU2, based on the web equivalent for Pantone 291C
  •  
    CU3, based on the web equivalent for Pantone 284C
  •  
    CC, based on the web equivalent for Pantone 292C
  •  
    CA, based on the web equivalent for Pantone 279C
  •  
    CY, based on the web equivalent for cyan
  •  
    AD, based on the web equivalent for Pantone 300U
  •  
    CU4, based on the web equivalent for Pantone 286C
  •  
    CD, based on the web equivalent for Pantone 295C

Text Colors

The custom themes can be used to set text color. Some text colors may not be easy to read on a light background color. In those cases, a darker background color should be assigned.
