Get Started

Step 1: Add Styles

Add the Tyfy for Bootstrap stylesheet to your project. You can add the compiled CSS to your HTML or import the SCSS file into your stylesheet.

Step 2: Add JavaScript

Add the following JavaScript resources to your HTML.

  • jQuery slim build
  • Popper.js
  • Bootstrap

Step 3: Add (Optional) Resources

I recommend two additional components.

  • webfontloader
  • Fontawesome 5
