Using TYBS

TYBS is a themed version of Bootstrap 4. It can be used, for the most part, as a drop-in replacement for standard Bootstrap. Therefore, all of the startup steps for Bootstrap also apply to TYBS. Please see the Bootstrap documentation on getting started. Simply replace the default Bootstrap CSS with the TYBS CSS file.

Key Differences

TYBS differs from Bootstrap in several key ways:

  • Space around and between elements is larger.
  • Font sizes are larger.
  • Font weights are a little heavier.
  • Line heights are larger.
  • The body font is serif, not sans-serif.
