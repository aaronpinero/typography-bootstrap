This customization of Bootstrap v4 applies the typographic styles, vertical rhythm, and grid stability from the Tyfy framework to Bootstrap's robust library of content and user interface patterns.

Specifically, the required and essential modules from the Tyfy framework are used to customize Bootstrap in a sustainable way, consistent with Bootstrap's theming guide. In addition, Bootstrap has been amended to include additional variables – fonts, colors – that are useful for the author's professional projects.
